About DDCcontrol

DDCcontrol is a tool used to control monitor parameters, like brightness and contrast, without using the OSD (On Screen Display) and the buttons in front of the monitor.

All the software provided by monitor manufacturers (NEC/Mitsubishi NaviSet and Samsung MagicTune) are closed-source and only runs on Windows. DDCcontrol is open-source and runs on Linux.

There is two versions of this tool, a command line version, and an user-friendly Gnome GUI.
