Appendix A. Report problems and successes

If you have problems building or running DDCcontrol, if your monitor doesn't work with DDCcontrol, or if your graphics card is not listed in the Appendix C, Supported graphics cards, please write a message to ddccontrol-users@lists.sourceforge.net.

For more informations on ddccontrol mailing lists, see the sourceforge project page.
