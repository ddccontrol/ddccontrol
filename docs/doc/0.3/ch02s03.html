Compile and install

To compile and install the source tarballs, type:

# ./configure
# make
# su root
Password: <root password>
# make install

You need to do this in ddccontrol and ddccontrol-db directories.

If you set the --prefix parameter when configuring one package, you must set it to the same value when configuring the other.
