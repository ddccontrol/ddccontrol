SiS

SiS graphics controllers only work with direct PCI memory access, so you don't have to have any kernel module built or loaded.

SiS chipset support table

These SiS chipsets have been reported to work. If yours is not listed, please send a message to one of the DDCcontrol mailing lists (see Appendix A, Report problems and successes).

dev PCI
SiS [M]661xX/[M]741[GX]/[M]760 PCI/AGP VGA Adapter No Yes
