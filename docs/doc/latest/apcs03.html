Intel

i810/i815/i830/i845/i855/i865/i915/i945 with Integrated Graphics Controller

Intel integrated graphics controllers only work with direct PCI memory access, so you don't have to have any kernel module built or loaded.

Intel chipset support table

These Intel chipsets have been reported to work. If yours is not listed, please send a message to one of the DDCcontrol mailing lists (see Appendix A, Report problems and successes).
